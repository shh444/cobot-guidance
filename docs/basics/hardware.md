# 하드웨어

:::{admonition} 이 장에서 배우는 것
:class: note
- UR 시스템을 이루는 기본 하드웨어 구성
- 초보자가 장비 앞에서 먼저 확인해야 할 것
- Tool, I/O, 네트워크를 문서에 어떻게 정리하면 좋은지
:::

## 명칭 익히기

유니버설로봇에서 주요 사용하는 명칭입니다.

| 구성품 | 역할 | 초보자가 처음 해야 할 일 |
| --- | --- | --- |
| 로봇암(Robot Arm) | 실제로 움직이는 로봇 팔 | 관절 방향과 작업 반경 감각 익히기 |
| 베이스(Base) | 로봇이 고정되는 구조물 | 설치 방향과 작업 반경 확인 |
| 툴플랜지(Tool Flange) | 툴이 붙는 면 | 툴 체결 상태와 TCP 기준 이해 |
| 앤드이펙터(End Effector) | 그리퍼, 툴, 스크류 드라이버 등 | 실제 작업 기능 확인 |
| 컨트롤러(Control Box) | 전원, 제어, I/O, 통신 중심 | 포트 위치와 전원 흐름 파악 |
| 티치팬던트(Teach Pendant) | 화면 조작과 teaching | 비상정지, 전원 버튼, 화면 구조 익히기 |

## End Effector 관점에서 하드웨어 보기

UR 입문자는 로봇 팔 자체보다 **앤드이펙터가 작업 품질을 좌우한다**는 점을 빨리 이해해야 합니다.

앤드이펙터가 바뀌면 보통 아래 항목을 다시 확인해야 합니다.

- TCP
- Payload
- CoG
- Tool I/O
- 제어 명령어

:::{tip}
툴 변경 시 반드시 "TCP / Payload / Cog"가 정확히 기입이 되어야 합니다.
:::

## Control Box에서 입문자가 알아야 할 것

Control Box는 단순한 전원 상자가 아니라 **제어와 통신의 중심**입니다. 초보자는 최소한 아래만 알아도 충분합니다.

- 전원 on/off 흐름 : 전원은 기본 220V 입력이지만 컨트롤러 타입에 따라 DC 입력도 가능합니다.
- Teach Pendant 연결 여부 : 티치 팬던트는 연결 해제가 가능하며 e-stop이 연결되어 있기 때문에 유무 설정을 팬던트에서 반드시 해야 합니다.
- Ethernet 연결 위치 : 컨트롤러 하단부 랜선 연결 포트가 있습니다.
- 일반 I/O와 안전 I/O의 구분 : 노란색은 안전 I/O, 회색,초록색은 일반 I/O 입니다.

## Teach Pendant에서 입문자가 먼저 익혀야 할 것

- 전원 버튼 : 로봇을 킬 수 있으며 필요 시 컨트롤러의 Remote 를 이용해서 전원을 키거나 끌 수 있습니다.
- 비상정지 : 팬던트는 안전 표준에 의해서 반드시 비상정지가 포함되어 있어야 합니다.
- 조그/Move 화면 : 로봇을 수동으로 움직이거나 뒤의 프리드라이브 버튼(검정색)으로 로봇을 수동 이동할 수 있습니다.
- Program Tree : 로봇 프로그램을 할 수 있습니다.
- 알람/로그 확인 위치 : Log 탭에서 문제가 발생한 내용 파악이 가능합니다.
