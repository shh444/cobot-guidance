# 유니버설로봇 구성

:::{admonition} 이 장에서 배우는 것
:class: note
- UR 기본 하드웨어 구성(로봇암, 컨트롤러, 팬던트)
- 필수 연결 요소(220V 전원 케이블, 로봇 연결 케이블)
- 팬던트 소프트웨어 `PolyScope`를 어떻게 이해하면 되는지
:::

## 한눈에 보는 기본 구성

유니버설로봇(UR)은 **하드웨어 3요소 + 케이블 2요소**로 구성 되어 있습니다.

| 구성 요소 | 역할 |
| --- | --- |
| 로봇암 (Robot Arm) | 6축 다관절 로봇, 실제로 움직이며 작업을 수행 |
| 컨트롤러 (Control Box) | 전원 공급, 모션/안전 제어, I/O 처리 |
| 팬던트 (Teach Pendant) | 작업자가 조작/티칭하는 인터페이스 |
| 220V 전원 케이블 | 컨트롤러 전원 입력 |
| 로봇 연결 케이블 | 컨트롤러와 로봇암 연결 |

:::{tip}
1. 기본은 AC 전원 공급이지만 DC OEM 컨트롤러를 사용하면 파워서플라가 바뀌어 DC 전원 공급으로 구동이 가능합니다.
2. 현장에서는 로봇 연결 케이블을 "동축 케이블"이라고 부르기도 하며 일반형 케이블와 가동형 케이블이 있습니다.
:::

## 어떻게 연결되는가

기본 연결 흐름은 아래처럼 보면 됩니다.

1. `220V 전원 케이블`로 컨트롤러에 전원을 공급
2. 컨트롤러와 로봇암을 `로봇 연결 케이블`로 연결 (커넥터를 시계 반시계로 끝까지 돌린 이후 강하게 두 케이블을 결합 후 시계방향으로 돌려줌)
3. 팬던트는 컨트롤러에 연결되어 있으며 프로그래밍을 할 수 있고 필요 시 제거할 수 있습니다.

즉, **컨트롤러가 중심**이고 로봇암과 팬던트가 여기에 연결되는 구조입니다.

## 팬던트 소프트웨어 PolyScope 이해하기

`PolyScope`는 팬던트에서 실행되는 UR 운영 소프트웨어입니다.  
쉽게 말하면 "로봇을 설정하고, 프로그래밍 하고, 실행하는 작업 화면"입니다.

리눅스 Debian OS이며 Polyscope 프로그래밍 설치되어 있습니다.

폴리스코프는 크게 아래와 같이 구성되어 있습니다.

| 구분 | 무엇을 하는 곳인가 | 대표 항목 |
| --- | --- | --- |
| Installation | 로봇의 설정 탭 | TCP, Payload, I/O, Safety, 좌표계 |
| Program | 로봇 동작 프로그램 탭 | Waypoint, MoveJ/MoveL, IF문, while문 |
| Run/Monitor | 실행 상태 확인 및 운전 | Run/Pause/Stop, 상태 메시지, 로그 |

## 첫 세팅 전 체크 포인트

- 로봇암 설치 방향과 작업 범위가 공정과 맞는가
- 콘센트에 220V 전원 접지가 잘 되어 있는가
- 로봇 연결 케이블 체결이 확실한가
- TCP, Payload, Safety 기본값을 실제 툴/작업물 기준으로 정확히 입력했는가
