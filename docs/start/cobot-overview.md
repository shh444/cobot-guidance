# 협동로봇이란?

:::{admonition} 이 장에서 배우는 것
:class: note
- 협동로봇을 기존 산업용 로봇과 어떻게 구분해서 이해하면 좋은지
- 협동로봇의 고유 장점이 무엇 인지
- 협동로봇 도입 시 고려해야 할 사항
:::

## 협동로봇 이란?

쉽게 말하면 협동로봇은 **사람 근처에서 함께 일하도록 안전 기능을 갖춘 산업용 로봇 시스템**입니다.  
핵심은 "로봇이 작다/가볍다"가 아니라, **필요한 안전 제어 시스템을 내장해 운용한다**는 점입니다.

표준 관점에서 보면 역할이 나뉩니다.

- `ISO 13849-1` : 안전 관련 제어부의 설계/검증 기준입니다. 안전 기능(정지, 속도 제한 등)이 필요한 성능수준(`PLr`)을 만족하는지 판단할 때 사용합니다.
- `ISO/TS 15066` : 사람-로봇 협업 작업 시 적용하는 실무 가이드입니다. 협업 운전 조건과 인체 접촉 관련 고려사항을 정할 때 기준이 됩니다.

즉, 협동로봇 안전은 감각이 아니라 **표준 기반 설계와 위험성 평가**로 정해지며, 그 기준에 맞춰 실제 운전 조건(속도·힘·거리·레이아웃)을 설정합니다.

## 전통 산업용 로봇과 협동로봇의 차이

| 항목 | 전통 산업용 로봇 | 협동로봇 |
| --- | --- | --- |
| 기본 설계 목표 | 고속·고강성·대량생산 | 사람 협업·유연 생산·빠른 전환 |
| 구조 특성 | 강건한 구조(무겁고 내구성 높음) | 상대적으로 경량 구조(설치/이동 유리) |
| 속도/출력 | 고속·고출력에 강점 | 협업 안전을 고려한 속도/힘 운용 |
| 프로그래밍 난이도 | 셋업/운용 복잡도가 높은 편 | 입문자가 배우기 쉬운 인터페이스 |
| 안전/공간 | 보통 펜스·안전구역 중심 | 위험성 평가 충족 시 인접 작업 가능 |

## 협동로봇 실무 장점

1. 산업용 로봇은 공정을 모두 자동화를 해야하지만, 협동로봇은 자동화 난도가 높은 구간은 사람이 맡고, 반복·정형 작업은 로봇이 맡은 후 **단계적으로 난도 어려운 부분 까지 자동화 범위를 넓힐 수 있습니다**.
2. 본체가 상대적으로 가벼워 부가축/베이스/브라켓 등 주변 설비 요구가 줄어들고, **설비의 전체 비용과 크기를 함께 줄이기 쉽습니다**.
3. 안전 요건을 충족해 펜스 의존도를 낮출 수 있는 공정에서는 **레이아웃 면적을 크게 줄이고 공정 배치를 유연하게 바꾸기 쉽습니다**.
4. AMR 연계, 사람-로봇 협업, 운송·설치 작업 등 **이동형/유연 생산 시나리오로 확장하기 좋습니다**.

## 협동로봇을 도입할 때 무엇을 고려해야 할까?

1. **작업 범위를 고려한 배치**: 로봇의 도달 반경, 간섭, 작업자 동선을 함께 보고 셀 레이아웃을 먼저 정해야 합니다.
2. **페이로드 결정**: 작업물 무게 + 툴 무게 + 케이블/어댑터 여유 하중까지 합산해 안전 여유를 포함한 페이로드를 정해야 합니다.
3. **툴 선정**: 로봇이 실제로 제어할 툴(그리퍼, 흡착, 스크류드라이버 등)의 인터페이스(I/O, 공압, 통신)와 제어 난이도를 미리 확인해야 합니다.
4. **목표 성능 정의**: 필요한 정밀도, 어플리케이션 품질, 사이클타임을 먼저 정하면 모델/툴/공정을 더 정확히 선정할 수 있습니다.
5. **안전 전략 수립**: 위험성 평가를 바탕으로 속도 제한, 안전 정지, 거리 감시, 펜스/센서 적용 범위를 결정해야 합니다.
6. **설비 연동성**: PLC, 비전, 컨베이어, MES/상위 시스템과 신호 연동이 필요하다면 초기에 협의하는 것이 좋습니다.
7. **운영/유지보수 체계**: 티칭 담당자 선정 및 교육, 문제 발생 시 대응책안, 백업/복구 절차를 정해 두면 도입 후 안정화가 빨라집니다.
